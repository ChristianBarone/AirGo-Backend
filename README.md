# AirGo-Backend

## 1. Requisitos mínimos del sistema
Para asegurar el correcto funcionamiento de la aplicación, el dispositivo debe cumplir con las siguientes especificaciones:
- Tener un teléfono móvil con sistema operativo Android 7.0 o superior. No es compatible con dispositivos iOS.
- Contar con una conexión a internet.
- Disponer de GPS en el dispositivo.

## 2. Enlace de descarga
Podéis descargar el archivo ejecutable .apk de la aplicación haciendo clic en el siguiente enlace:

https://drive.google.com/file/d/1osCMNdts6wXAqG4te-Q4JCu_q-mtVKGv/view?usp=drive_link

## 3. Instalación y ejecución
1. Abre el enlace de Google Drive desde tu dispositivo Android.
2. Selecciona la cuenta de correo con la que quieras abrirlo y elige la opción “Instalador de paquetes”.
3. Espera a que se complete la descarga del archivo AirGo.apk (Nota: En este punto, el proceso de instalación puede quedarse atascado. Si eso pasa, simplemente sal de la aplicación y vuelve a repetir los pasos 1 y 2).
4. Al intentar abrir el archivo por primera vez, Android mostrará un aviso de seguridad similar a: “Por tu seguridad, actualmente el teléfono no puede instalar aplicaciones desconocidas de esta fuente. Puedes cambiarlo en Configuración”. Pulsa el botón “Configuración” que aparece en el mismo aviso.
5. Activa el interruptor “Conceder permiso” para Google Drive.
6. Vuelve atrás y pulsa el botón “Instalar”.
7. Una vez finalizado el proceso, el sistema mostrará una notificación confirmando que la aplicación se ha instalado correctamente. Puedes pulsar “Abrir” directamente desde ese aviso o, si no, buscar el nuevo icono de AirGo en el menú de inicio del teléfono para iniciarla.

## 4. Permisos del sistema
Para el correcto funcionamiento de AirGo, se deben conceder dos permisos del sistema:
- Permiso de Notificaciones: Al entrar en la aplicación, el sistema te pedirá autorización para enviarte notificaciones. Pulsa “Permitir”
- Permiso de Ubicación: La primera vez que navegues a la pantalla de “Actividad”, la aplicación necesitará acceder al GPS. Cuando aparezca el aviso de Android, selecciona “Mientras se usa la aplicación”.

## Con estos pasos, la aplicación estará configurada y lista para crear rutas saludables!
